# vuepress-plugin-live2d-model

## How to use

1. install `npm i vuepress-plugin-live2d-model -S`

2. add in `config.js`

```js
module.exports = {
  plugins: [
    "vuepress-plugin-live2d-model",
  ]
}
```

## Tip

- This plugin supports `vuepress 1.x`
- This plugin will randomly change the white cat or black cat when router path change.
